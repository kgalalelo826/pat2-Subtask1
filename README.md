# Morse Code Research

##  Overview
Morse code is a method of communication that uses a series of dots (.) and dashes (-) to represent letters, numbers, and symbols. It is one of the earliest forms of digital communication and is still used today in certain fields like aviation, emergency communication, and amateur radio.



## History of Morse Code
Morse code was developed in the 1830s and 1840s by Samuel Morse and Alfred Vail. It was originally created for use with the telegraph system, allowing messages to be sent over long distances using electrical signals.

The first official Morse code message was sent in 1844 and read:
**"What hath God wrought"**



##  How Morse Code Works
Morse code uses:
- **Dots (.)** → Short signals  
- **Dashes (-)** → Long signals  

Each letter and number has a unique combination of dots and dashes.

### Examples:
| Letter | Morse Code 
 A        .-        
 B        -...      
 C        -.-.      
 1        .----     
 2        ..---     

### Spacing Rules:
- Space between parts of the same letter → Short pause  
- Space between letters → Medium pause  
- Space between words → Long pause
  

##  Example Words in Morse Code

- HELLO → `.... . .-.. .-.. ---`
- SOS → `... --- ...`
- CODE → `-.-. --- -.. .`


##  Uses of Morse Code Today
- Emergency distress signals (e.g., SOS)
- Aviation communication
- Military communication
- Amateur (ham) radio
